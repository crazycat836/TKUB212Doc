## 架構



### 後端
------------

使用 Node.JS + Express 在本機測試

用 Google Firebase 做 Hosting

### 前端
------------

採用 HTML + CSS + JavaScript 去建構整個頁面

HTML：Pug 模板

CSS：Materialize

JavaScript：Jquery

Google Font API：Material icons、中文字型－思源黑體

Gulp：Pug 模板 轉成 HTML

Webpack：將 CSS, img, JavaScript 打包，壓縮 JavaScript, CSS，加速載入速度，產生靜態內容
